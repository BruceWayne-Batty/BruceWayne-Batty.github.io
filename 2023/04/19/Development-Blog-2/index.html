Development Blog 2

This is the devlopment blog for game “TechnoScourge: Rise of the Adaptivethe” in the 3rd and 4th week(Since there is one week holiday in the 3rd week so I combine the blog of these two weeks together)

There is some problems with using scriptable object as the user stat directly, we need to use a monobehaviour script to store the data of the player, so an additional script for the player stat is needed.

For the inventory system, I implemented a simple system which now can add or remove items in the in game ui. The items are saved as scriptable object and the inventory system will only read the data in it instead of changing anything like the way we do in the player stat, so it will not have any problem using the scriptable item.

I connected the inventory system with player stat now, and the stat is not shown in the in-game UI, but the icons of the items does. I will show the user stats in the panel and also the equipment gears on the panel.

Now I’m working on the inventory part of the GDD file, the plan now is to implement at least 2 different builds in the demo, and design at least 8 builds for the game.
