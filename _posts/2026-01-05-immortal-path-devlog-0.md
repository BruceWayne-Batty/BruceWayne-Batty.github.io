# 🌱 缘起
最近读完了一本修仙题材的番茄小说，读的时候就在想——书里那些历练、突破、宗门争斗的情节，如果做成游戏应该会很有意思吧。

起初想过直接借用小说的剧情，但转念一想，小说里那个穿越过去的主角，其实挺像 Kenshi 里那种从零开始的“无名之辈”的。于是念头一转：不如自己做一个仙侠版的开放世界游戏，带点 Kenshi 那种自由与挣扎的味道。

不过现实是：美术不是我的强项（可以说完全不会画）。所以很快决定，用文字冒险的形式来实现这个想法。这也是我初高中时躲在被窝里玩手机最爱的那种游戏类型——纯粹靠文字和选择来推进剧情，反而能留给玩家更多想象空间。

当然，如果将来有时间，或许会补上一些简单的动画和图标。但眼下，文字就是我的画笔，代码就是我的灵脉。

# 📅 计划（大概）
目前暂定的开发周期是 1-3 个月。不想把战线拉得太长，怕自己失去动力。
目标是每周至少推进 3 次开发进度，并尽量保持写日志的习惯。

当然，计划归计划，实际做起来难免会调整。如果真的变了……我就回来改这一条 😄
